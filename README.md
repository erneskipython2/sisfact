# sisfact
Sistema de facturación básico con gestión de inventario, clientes, usuarios y reportes generales. Base de datos Mysql.

## Prestaciones
- Gestion de clientes
- Gestion de Inventario
- Gestion de Usuarios
- Facturacion
- Validacion QR
- Graficos dinamicos 
- Reportes


## Detalles tecnicos
- 100 % php y html
- Codigo limpio
- MVC
- MySql
